# AI WhatsApp Reply Automation

A Python-based automation project that reads visible WhatsApp chats, generates contextual replies using the Groq API, and automatically pastes and sends the response.

## Features

- Reads visible WhatsApp chat
- Generates AI-powered contextual replies
- Uses Groq API for response generation
- Automatically pastes and sends the reply

## Tech Stack

- Python
- PyAutoGUI
- Pyperclip
- Groq API
- python-dotenv

## Installation

```bash
pip install -r requirements.txt
```

## Note

This project uses screen coordinates for automation. Coordinates may need to be adjusted depending on screen resolution and display settings.
